## 01 · Brand Introduction

**NINE ONE** 是一家从新西兰起步的未来运动基础设施平台公司。

**名字来源：** 创始人的两个孩子 — 老大名字含「一」，老二名字含「九」。1 + 9 = 10，在中国文化中「十」代表十全十美。Nine One 代表家庭、责任、长期主义与传承。

**总部：** Auckland, New Zealand (North Shore)
**扩张方向：** NZ → Australia → Asia Pacific
**网站：** nineonesports.co.nz

---

## 02 · Brand Philosophy

NINE ONE 不是传统运动品牌，也不是冰冷的 AI 科技公司。

它是一个 **Future Movement Infrastructure System** — 将运动空间、社区与智能系统连接起来的生态平台。

**核心哲学：**
- **Calm Movement** — 克制的运动表达
- **Connected Intelligence** — 连接的智能系统

**品牌气质关键词：** Calm · Premium · Spatial · Human · Intelligent · Connected · Future-facing · International

## 05 · Logo System

### The Mark
- **字形：** `no` — 小写字母 n + o
- **n：** 圆角 U 形，底部开口，等宽圆润笔画
- **o：** 圆角矩形描边，四角大圆角
- **字体：** 品牌定制字形，基于 Satoshi / General Sans 骨架

### Wordmark
```
no
nine one
```

### 子品牌结构
```
no          no
nine one    nine one
SPORTS      TECHNOLOGIES
```
子品牌名使用 **Josefin Sans 400**，letter-spacing 0.45em

### Logo 颜色规范

| 场景 | Mark 颜色 | 背景 |
|------|-----------|------|
| Master · Dark | White `#F5F5F3` | Black `#0A0A0A` |
| Master · Light | Black `#0A0A0A` | White `#F5F5F3` |
| Sports · Dark | Lime `#C7F000` | Dark `#080900` |
| Technologies · Dark | Blue `#2D7DFF` | Navy `#050D1A` |
| Transparent | White / Black | 透明底 |

### Safe Area 安全间距
- 基础单位 **X** = 字母 O 的高度
- Logo 四周最小保留 **1X** 安全间距

### 最小尺寸

| 类型 | Digital | Print |
|------|---------|-------|
| Full Logo | 120px | 25mm |
| Icon Only | 24px | 8mm |

### 禁止用法
- ✗ 使用数字 9 或 1 作品牌图形
- ✗ 拉伸、压缩、旋转 Logo
- ✗ 添加阴影、描边、发光特效
- ✗ 改变字距或结构
- ✗ Esports / gaming aesthetics
- ✗ AI cliché graphics

## 07 · Typography 字体系统

> **核心原则：System first. Style second.**

### 字体层级结构

| Layer | 字体 | 角色 |
|-------|------|------|
| Core System | **Satoshi** | 主字体，覆盖所有品牌场景 |
| Accent | **Josefin Sans** | 辅助展示字体，限定使用 |
| Logo/Wordmark | **Custom Condensed** | 品牌标识，基于 Druk / Monument 方向定制 |

---

### 主字体 · Satoshi

**NINE ONE 核心品牌字体正式定义为 Satoshi。**

覆盖所有场景：Body text · Navigation · Buttons · UI · Deck · Docs · Captions · Data · Website · Product · Social · Motion

**为什么选 Satoshi：**
- Performance-driven，极强可读性
- 干净的几何比例，现代结构
- 足够科技但不过度未来感
- 极强品牌延展性，适合全系统部署
- 对标方向：Nike · Linear · Arc · Notion

**推荐字重：**

| 用途 | 字重 |
|------|------|
| Body text | Regular 400 |
| Navigation · UI | Medium 500 |
| UI emphasis | Bold 700 |
| Hero · Numbers | Black 900 |

---

### 辅助字体 · Josefin Sans

**定义为 Accent Typeface，不是主系统字体。**

只能用于：Campaign headline · Editorial moment · Slogan · Large visual statements · Artistic compositions

禁止用于：正文 · UI · App · Deck内容页 · 数据信息 · 长文本

---

### Logo / Wordmark 方向

基于 Condensed grotesk 骨架进行二次定制，方向参考：Druk Type Family · Monument Extended · Helvetica Now Display

---

### 字号层级

| 层级 | 字体 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|------|
| H1 Hero | Satoshi | 72–96px | 900 | 100–110% | 首页主标题 |

| Campaign Title | Josefin Sans | 按场景 | 600 | 按需 | Editorial only |

## 11 · Motion 动效规范

### 动效哲学
> **Calm Motion** — 动效应该像呼吸，而不是表演。

### Logo 动效
**允许：** Fade in · Slow reveal · Connected line motion · Subtle opacity transition · Spatial transition

**禁止：** Bounce · Rotation · Gaming animation · Aggressive speed · Glitch · Explosion

### Transition Duration

| 类型 | 时长 |
|------|------|
| UI hover | 200–300ms |
| Page transition | 500–800ms |
| Hero reveal | 1.2–2s |
| Logo intro | 1–1.5s |
